# Spell compendium

**Branch:** feature/spells · **Date:** 2026-07-21 · **Author:** Denis (+ Claude)

## What changed
New "Spells" tab in the sidebar (under Bestiary): the full Open5e spell
list (~1,435 spells, SRD + CC sources) in the same table-with-filters
style as the Bestiary. Filters: text search, level (cantrip–9),
school, class, source; concentration/ritual badges in the list; each
spell opens a formatted detail page (casting time, range, components,
duration, classes, full description, at-higher-levels). New `spells`
table using the shared-layer pattern (`user_id NULL` = Open5e base),
loaded by the idempotent `scripts/import-open5e-spells.mjs`.

## Why
Spell lookups were the last thing still sending the DM to 5e.tools
mid-session.

## How to test
`node scripts/import-open5e-spells.mjs`, then Spells in the sidebar →
filter level 3 + Evocation + Wizard (Fireball should be in the ~24
results) → open it and check the formatted page.

## Deploy steps
Run once on the server after deploy:
`docker compose exec nilbot node scripts/import-open5e-spells.mjs`
(idempotent; loads the shared spell layer into the production DB).
